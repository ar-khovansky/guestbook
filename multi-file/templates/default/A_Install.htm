Логин До 30 символов; только латинские буквы, цифры и '_'
Пароль повтор 5-16 символов; только латинские буквы, цифры и '_'

- Параметры будут сохранены в файл config.php в корневой директории приложения.
- Загружаемые файлы изображений сохраняются в files/.
- Логи в отладочном режиме (define('DEBUG', true);) сохраняются в logs/.
PHP должен иметь права записи в эти директории.
